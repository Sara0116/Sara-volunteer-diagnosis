# 志愿诊断网页原型

这是第一版本地网页原型，不需要安装 Node.js、Python、数据库或服务器。

## 怎么打开

直接双击打开：

```text
index.html
```

或者用 Chrome 打开这个文件。

## 现在能做什么

- 输入分数、位次、选科
- 选择偏好：综合、就业、考研、考公
- 使用示例志愿数据自动诊断
- 上传 Excel 志愿表
- 生成原表分析
- 生成建议调整后排序
- 展示综合建议分
- 展示学校层次、就业、考公考研分析
- 展示来源依据
- 导出 Excel 报告

## 现阶段的限制

- Excel 解析和导出使用本地 `vendor/xlsx.full.min.js`，上线时记得一起上传 `vendor` 文件夹。
- 评分规则是第一版原型规则，还不是最终商业算法。
- 学校层次和专业标签目前写在 `app.js` 里，后面可以改成数据库。
- 导出的 Excel 暂时不做复杂样式，只先保证数据结构。

## 什么时候需要安装环境

现在不需要。

后面如果要做正式工程化版本，再安装：

- Node.js：用于前端项目工程化
- Python：用于后端和 Excel 高级处理
- SQLite/PostgreSQL：用于保存学校、专业、投档线和来源数据
